# Contributing to YM Tech Labs

Thank you for helping improve practical IT education.

## Before You Contribute
- Check existing issues before opening a new one.
- Never commit credentials, private IP addresses, client data, or licensed materials.
- Test labs from a clean environment and document prerequisites.

## Pull Requests
- Keep each pull request focused on one change.
- Include clear documentation and verification steps.
- Use descriptive commit messages.

## Conduct
Be respectful, constructive, and professional in all discussions.
